# J.A.R.V.I.S.A.
локальная клиент–серверная платформа для взаимодействия с искусственным интеллектом. Проект позволяет работать с локальными языковыми моделями и генерацией изображений без зависимости от облачных сервисов.

Основной интерфейс реализован в виде веб-приложения (mobile.html), доступного с компьютеров и мобильных устройств. Серверная часть написана на Python с использованием Flask и отвечает за обработку запросов, работу с моделями и анализ данных. Дополнительно реализованы интерфейсы для работы с кодом (coder.html) и Telegram-бот как вспомогательный способ взаимодействия.

Проект поддерживает генерацию текста, анализ файлов, генерацию изображений, хранение истории чатов и удалённый доступ через CloudPub. J.A.R.V.I.S.A. работает на Windows, macOS и Linux.

Основные возможности

работа с локальными LLM через Ollama

веб- и мобильный интерфейсы

генерация изображений (Stable Diffusion)

анализ текстовых файлов и PDF

дополнительный Telegram-бот

кроссплатформенность

Назначение

Проект разработан в образовательных целях и может использоваться для изучения принципов работы локальных нейросетей, веб-разработки и клиент–серверной архитектуры.
